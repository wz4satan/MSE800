# Mathematical Operations Program - Week 2 Activity 1

## Project Overview
This project is a Python-based mathematical tool developed for **Week 2 - Activity 1**. It performs basic arithmetic operations and advanced calculations (like factorials) while supporting multiple data types, including Integers, Floats, and **Complex Numbers**.

The program is designed with modularity in mind, utilizing four distinct functions to handle logic, input, and execution.

## Features
- **Basic Arithmetic**: Supports Addition (`+`), Subtraction (`-`), Multiplication (`*`), Division (`/`), and Modulo (`%`).
- **Complex Number Support**: Fully compatible with Python's complex data type (e.g., `2+3j`).
- **Factorial Calculation**: Calculates the factorial for non-negative integers.
- **Robust Input Handling**: Includes a custom input function that validates user data and handles different numeric types.
- **Error Handling**: Gracefully manages "Division by Zero" and invalid data type inputs.

## Requirements
- Python 3.x
- No external libraries required (uses built-in `math` module).
