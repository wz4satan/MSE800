# Land Dimension Calculator (OOP Project)

This project is a Python-based application designed to calculate the area and perimeter of a rectangular piece of land. It is built using Object-Oriented Programming (OOP) principles, demonstrating class definition, encapsulation, and modular code structure.

## Project Overview
The application prompts the user to input the length and width of a piece of land. It then instantiates a `Land` object and utilizes its methods to perform calculations and display the results.

## Features
- **Object-Oriented Design**: Logic is encapsulated within a `Land` class.
- **Modular Structure**: The project is split into a class module and an execution script.
- **User Interaction**: Accepts real-time numerical input from the user.
- **Error Handling**: Includes basic validation to handle non-numeric inputs.

## Project Structure
- `class_Land.py`: Contains the `Land` class definition with methods for area, perimeter, and dimension display.
- `main.py`: The main entry point of the application that handles user input and object instantiation.

## Prerequisites
- Python 3.x installed on your system.

## How to Run
1. Download or clone the project files.
2. Ensure both `class_Land.py` and `main.py` are in the same folder.
3. Open your terminal or command prompt.
4. Navigate to the project directory.
5. Run the following command:
   ```bash
   python main.py
